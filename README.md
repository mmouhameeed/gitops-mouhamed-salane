# Projet GitOps Observable - Mouhamed Salane

Plateforme GitOps observable sur Kubernetes (cluster 3 nœuds via k3d).

## Composants
- Application : Flask (dossier `app/`)
- CI : GitHub Actions
- Registre : GitHub Container Registry (ghcr.io)
- GitOps : Argo CD
- Observabilité : Prometheus, Grafana, Loki
